# Wellpass Analyse

Browserbasierte Auswertung von EGYM Wellpass CSV-Exporten.
Die Verarbeitung laeuft lokal im Browser, es werden keine Daten hochgeladen.

## Features

- CSV-Upload per Klick oder Drag-and-drop
- Kosten-Nutzen-Auswertung fuer den gewaehlten Zeitraum
- Check-ins je Einrichtung mit anpassbaren Tageskartenpreisen
- Monatlicher Verlauf (auch ueber mehrere Jahre)

## Sprachunterstuetzung

- Unterstuetzt werden aktuell CSV-Exporte auf Deutsch und Englisch.
- Andere Sprachen sind derzeit nicht unterstuetzt.
- Wenn dein Export in einer anderen Sprache ist, stelle die App-Sprache vor dem Export auf Deutsch oder Englisch um.

## Quickstart: CSV aus der Wellpass App exportieren

1. Wellpass App oeffnen.
2. Zu `Profil` wechseln.
3. `Check-in Berichte` auswaehlen.
4. Zeitraum auswaehlen (empfohlen: gesamter Mitgliedschaftszeitraum).
5. `Export` tippen und die Datei als `.csv` speichern oder teilen.
6. Diese Seite oeffnen und die CSV per Drag-and-drop hochladen oder anklicken.

Hinweis: Je mehr Monate/Jahre im Export enthalten sind, desto genauer ist die Kosten-Nutzen-Auswertung.
